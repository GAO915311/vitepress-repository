## 回调

在调用一个函数之后，需要在函数执行中或执行后，将执行结果再传递给调用者（客户端）并进行一系列后续操作时，可以使用回调机制。通常是：
1. 执行某个操作需要耗时，异步执行后进行回调。
2. 调用者不再关心回调函数中进行的后续操作。
3. 程序需要监听函数中某个动作的完成，从而进行下一步操作。
